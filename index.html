Python Вступ

Python — це популярна мова програмування. Вона була створена Гвідо ван Россумом і випущена в 1991 році.

Навіщо використовувати Python?

  • Python працює на різних платформах (Windows, Mac, Linux).
  • Має простий синтаксис, схожий на англійську мову.
  • Дозволяє писати програми з меншою кількістю рядків коду.
print("Привіт, Світе!")

Python Синтаксис

Синтаксис Python можна виконувати безпосередньо в командному рядку або створивши файл з розширенням .py.

Відступи в Python

Відступи (пробіли на початку рядка) є надзвичайно важливими. Python використовує відступи для визначення блоку коду.

if 5 > 2:
  print("П'ять більше двох!")

Python Змінні

Змінні — це контейнери для зберігання значень даних.

x = 5
y = "Ivan"
print(x)
print(y)

На відміну від інших мов, Python не має команди для оголошення змінної. Вона створюється в момент першого присвоєння їй значення.
