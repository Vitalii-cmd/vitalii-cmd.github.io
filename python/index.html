Що таке C++?

C++ — це кросплатформна мова програмування для створення високопродуктивних програм.

Вона була розроблена Б'ярном Страуструпом як розширення мови C і дає контроль над системними ресурсами.

Простий приклад

#include <iostream>
int main() {
  std::cout << "Привіт, Світ!";
  return 0;
}
